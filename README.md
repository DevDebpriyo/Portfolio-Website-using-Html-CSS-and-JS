# Portfolio Website 🌐  

This is a sample portfolio website I built to test my skills and add as a project to my resume. Built using **HTML, CSS, Tailwind CSS**, and hosted on **Netlify**.

- Interactive UI with Tailwind CSS  
- Social media links  
- Contact form  

## 📂 Technologies Used  
- **HTML**  
- **CSS** (Tailwind CSS)  
- **JavaScript**  
- **Netlify** (for hosting)
